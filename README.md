# Java Collections Practice

This repository contains beginner-friendly Java programs using the Java Collections Framework with user input. It covers ArrayList, HashSet, and HashMap along with placement-oriented coding practice.

## Programs Included

### HashMap Programs
- Character Frequency
- Word Frequency
- Student HashMap
- Search Student
- Update Student
- Delete Student

### HashSet Programs
- Remove Duplicates

### ArrayList Programs
